## 文档

* [Angular](https://angular.cn/docs)

## 环境

* node.js

## 使用

```bash
  #安装依赖
  npm install
  ionic serve #启动本地浏览器调试
  #常用指令 看ionic-cli（command List）
  #generate,build,cordova...
```
